# Starter for webpack and three.js

## Initialize source code
```bash
# Webpack and friends
$ npm i -D webpack webpack-dev-server html-webpack-plugin

# THREE.js
$ npm i three

# Initialize git repo
$ git init
$ git add .
$ git commit -m 'Initial commit'
```

## Development
```bash
$ npm i
$ npm start
```

## Clone
```bash
$ git clone https://github.com/11o9/threepack.git
$ git pull

$ npm i
```
